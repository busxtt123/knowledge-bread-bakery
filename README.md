# Knowledge Bread Machine｜知识面包机

![知识面包](assets/knowledge-bread-cover.png)

把你想读却没时间读透的书，定制成一门能学完、能记住、能用上的知识课程。

“知识面包机”吸取了《得到品控手册 9.0》里关于“做内容”的经验，也借用了《哆啦 A 梦》记忆面包的想象：把一本书重新做成高营养、易吸收、适合你当前需要的知识面包。

它会先采访你，听清阅读目的、可用时间和想解决的问题，再把书里的材料重组成类似得到课程的知识成品。让你在工作、学习和生活都很忙的时候，仍然有心力吸收高质量知识。

## 核心价值

**按需知识**：制作前先采访你。你可以定制课程的阅读时长、重点问题、使用场景和吸收节奏，少做筛选，直接得到适合自己的知识成品。

**大部头变课程**：把厚书、报告和复杂资料转化成可学习的课程。你身边多一个知识服务专家，不用和推荐算法博弈，也能持续接触远高于自媒体密度的信息。

**高营养、易吸收**：保留书里的关键营养，同时重新安排入口、顺序和讲述节奏。不是把知识变浅，而是让你更容易进入、理解和带走。

## 怎么开始

把本地文件路径发给 agent，说一句：

```text
请你将 /path/to/book.pdf 这本书转变为课程。
```

也可以更具体：

```text
请使用 knowledge-bread-machine，把 /path/to/book.pdf 做成午休 30 分钟能读完的知识面包。
```

你不需要先写大纲。给它材料，告诉它你的阅读目的，它会继续追问必要信息，然后开始制作。

## 适合做什么

- 一本书的 30 分钟知识课程。
- 3-5 讲的小课程。
- 手机阅读 PDF。
- 围绕某个问题的阅读路线，比如“这本书对我的管理判断有什么帮助”。
- 报告、讲义、课程草稿和资料包的课程化整理。

## 注意事项

该 skill 采用精细化的写作编排。每个任务通常需要运行 20 分钟到 3 小时不等，请使用高稳定的编程套餐或低价 API 服务。推荐模型：DeepSeek V4 Pro / Lite。

## 支持材料

- PDF、扫描 PDF、EPUB。
- Word、Markdown、TXT。
- 报告、讲义、课程草稿。
- 已整理成文件的资料包。

它不抓取网页，不转换公开视频、微信公众号、YouTube 或其他在线链接。请先把材料准备成本地文件。

## 安装

把仓库地址发给 agent，让它安装这个 skill：

```text
https://github.com/busxtt123/knowledge-bread-machine
```

如果 agent 需要 SSH 地址：

```text
git@github.com:busxtt123/knowledge-bread-machine.git
```

安装后，把材料路径交给 agent，就可以开始做知识面包。

## 扫描 PDF

普通 PDF、EPUB、Word、Markdown、TXT 通常可以直接处理。

扫描 PDF 需要 OCR。PDF OCR 需要本机安装识别工具；如果环境还没准备好，agent 会按提示指导你安装依赖工具。

识别质量太差时，OCR 结果不会进入后续创作。
